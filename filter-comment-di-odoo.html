Langkah-langkah Menambahkan Filter Comment di Odoo

Tutorial untuk pengguna Odoo

1. Aktifkan Debug Mode

Masuk ke Odoo. ketika di url dengan menambahkan ?debug=1, lalu muncul Gambar kepik di atas.

2. Navigasi ke Model res.partner

Buka menu Contacts yang berkaitan dengan model res.partner. Klik pada Edit SearchView untuk mengedit tampilan pencarian.

3. Edit Search View

Di bagian Inherited View, add line untuk membuat view baru

4. Buat Inherited View

Di bagian Inherited Views, klik untuk membuat inherited view baru. Isi informasi berikut:

  • View Name: Beri nama untuk inherited view, misalnya Inherit Comment Filter.
  • View Type: Pilih Search.
  • Active: Pastikan opsi ini dicentang.
  • Inherited View: Pastikan ini dicentang juga.

5. Base View

Model Data: Pilih (auto default)res.partner.select. External ID: (di langkah selanjutnya)biarkan kosong saja dulu

6. Menambahkan Kode XML

<xpath expr="//search" position="inside">
    <filter name="comment" string="Notes" context="{'group_by': 'comment'}"/>
</xpath>

7. Menyimpan dan Mengupdate

Setelah menempelkan kode, klik Save dan segarkan halaman Odoo.
