Panduan Transkripsi Audio Menggunakan AssemblyAI

  1. Siapkan File MP3
  2. Daftar di AssemblyAI dan dapatkan API Key Anda.
  3. Salin API Key yang diberikan.
  4. Gunakan Template Kode: 
    import assemblyai as aai

# Ganti dengan API key Anda
aai.settings.api_key = "API_KEY_ANDA"

# URL dari file yang ingin ditranskripsi
FILE_URL = "C:\\Users\\abajo\\Music\\1 - adhigana requirement (mp3cut.net).mp3"

# Konfigurasi pengaturan transkripsi
config = aai.TranscriptionConfig(language_code='id', speech_model=aai.SpeechModel.nano)

# Buat Transcriber dengan konfigurasi yang ditentukan
transcriber = aai.Transcriber(config=config)

# Transkripsi file audio
transcript = transcriber.transcribe(FILE_URL)

if transcript.status == aai.TranscriptStatus.error:
    print(transcript.error)
else:
    print(transcript.text)
  5. Simpa file sebagai transcribe.py.
  6. Gunakan Command Prompt (CMD) dan jalankan: 
    C:\Users\abajo\assemblyai-example>py transcribe.py

Persiapan Instalasi

  1. Instal Python - Download dan instal Python untuk Windows di sini.
  2. Perbarui Pip - Jalankan perintah berikut di Command Prompt: 
    C:\Users\abajo>py -m ensurepip --upgrade
  3. Cek Versi Pip - Periksa versi pip: 
    C:\Users\abajo>py -m pip --version
  4. Instalasi AssemblyAI - Jalankan: 
    C:\Users\abajo>py -m pip install -U assemblyai
